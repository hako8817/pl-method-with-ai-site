# 제안서 작성

수주형 프로젝트의 앞단에서 제안서를 기획 산출물로 다루는 방법론이다.

이 섹션은 `RFP 분석 -> 요구사항 대응 구조화 -> 제안 전략 -> 제안서 작성` 흐름을 중심으로, 제안서 작업을 단순 문서 작성이 아니라 앞단 기획 산출물로 보는 관점을 정리한다.

## 이 섹션에서 보는 것
- 제안서 작업의 앞단 구조
- 제안 자동화와 반자동화의 적용 지점
- 착수와 기획으로 넘어가는 입력 구조

2. [한장 요약](one-page-summary)
3. [목차 초안](toc-draft)
