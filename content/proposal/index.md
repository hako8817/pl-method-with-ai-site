# 게시용-Proposal-index-초안-v1

## 목적
이 문서는 Quartz의 `proposal` 섹션 index에 넣을 실제 게시용 초안이다.

## 초안
# 제안서 작성

수주형 프로젝트의 앞단에서 제안서를 기획 산출물로 다루는 방법론.

이 섹션은 `RFP 분석 -> 요구사항 대응 구조화 -> 제안 전략 -> 제안서 작성` 흐름을 중심으로,
제안서 작업을 단순 문서 작성이 아니라 앞단 기획 산출물로 보는 관점을 정리한다.

## 이 섹션의 역할
- 제안서 작업의 앞단 구조를 정리한다.
- 제안서 자동화와 반자동화의 적용 지점을 설명한다.
- 이후 착수와 기획으로 넘기는 입력 구조를 다룬다.
