# 게시용-Home-index-초안-v1

## 목적
이 문서는 Quartz 홈 화면에 넣을 실제 게시용 초안이다.

## 초안
# PL method with ai

기획자의 전체 프로세스를 `제안 -> 기획 -> PKM -> Knowledge` 관점으로 문서화한 방법론 아카이브.

이 사이트는 `전체프로세스-허브`, `proposal with AI`, `planning with AI`, `PKM OS for PM-PO`, `Knowledge`를 중심으로,
기획자가 실제 산출물과 업무 흐름을 어떻게 구조화하는지 보여주는 공개 허브다.

## 이 프로젝트가 다루는 것
- 기획자의 전체 프로세스를 문서화한다.
- 산출물 체인을 기준으로 제안과 기획을 다시 본다.
- 이후 시스템으로 확장 가능한 수준까지 구조를 고정한다.

## 읽기 시작 순서

2. [제안서 작성](proposal/)
3. [기획 방법론](planning/)

## 섹션
### 전체 프로세스
세 권의 흐름과 공통 기준을 연결하는 전체 지도.

[전체 프로세스](process/)

### 제안서 작성
수주형 프로젝트의 앞단에서 `RFP 분석 -> 제안 전략 -> 제안서 작성`을 다루는 방법론.

[제안서 작성](proposal/)

### 기획 방법론
산출물 체인을 기준으로 기획 문서와 업무 흐름을 다시 설계하는 방법론.

[기획 방법론](planning/)

### PKM
개인지식관리를 메모 습관이 아니라 운영체계로 보는 방법론.

[PKM](pkm/)

### Knowledge
기획자의 비즈니스기획, 디자인, 기술 지식을 구조화해 둔 지식 베이스.

[Knowledge](knowledge/)
