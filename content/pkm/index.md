# PKM

개인지식관리를 메모 습관이 아니라 운영 체계로 보는 방법론이다.

이 섹션은 기획자가 개인지식관리로 무엇을 어떻게 관리해야 하는지를 중심 질문으로 두고, 책의 구조와 대표 흐름 문서를 먼저 보여준다.

## 이 섹션에서 보는 것
- PKM을 운영 방법으로 보는 관점
- 책의 질문과 전체 구조
- 대표 흐름 문서를 통한 최소 공개 범위

## 범위 메모
- 현재 v1에서는 허브와 대표 구조 문서만 먼저 연다.
- 상세 장 원고와 집필 워크플로우는 이후 단계에서 공개 여부를 판단한다.
