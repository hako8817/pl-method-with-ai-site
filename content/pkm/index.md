# 게시용-PKM-index-초안-v1

## 목적
이 문서는 Quartz의 `pkm` 섹션 index에 넣을 실제 게시용 초안이다.

## 초안
# PKM

개인지식관리를 메모 습관이 아니라 운영체계로 보는 방법론.

이 섹션은 기획자가 개인지식관리로 무엇을 어떻게 관리해야 하는지를 중심 질문으로 두고,
책의 구조와 대표 흐름 문서를 먼저 보여준다.

## 이 섹션의 역할
- PKM을 저장소 소개가 아니라 운영 방법으로 설명한다.
- 책의 질문과 구조를 먼저 제시한다.
- 실제 데이터 전체 공개 대신 대표 흐름 문서만 먼저 연다.
